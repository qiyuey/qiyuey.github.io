Harness Engineeringпјҡз»ҷ Coding Agent еҘ—дёҠзј°з»і

Updated: 23 Mar, 2026
|

2026 е№ҙ 2 жңҲпјҢOpenAI еҸ‘дәҶдёҖзҜҮе·ҘзЁӢеҚҡе®ў Harness Engineering: Leveraging Codex in an Agent-First WorldпјҢи®ІдәҶдёҖдёӘдёүдәәеӣўйҳҹз”Ё Codex д»Һз©әд»“еә“ејҖе§ӢгҖҒйӣ¶дәәе·Ҙзј–з ҒгҖҒдә”дёӘжңҲй•ҝеҮәзҷҫдёҮиЎҢд»Јз Ғдә§е“Ғзҡ„ж•…дәӢгҖӮж–Үз« ж Үйўҳз”ЁдәҶвҖқHarness EngineeringвҖқпјҢдҪҶжӯЈж–ҮеҸӘжҸҗдәҶдёҖж¬ЎвҖқharnessвҖқиҝҷдёӘиҜҚпјҢжӣҙеғҸжҳҜдәӢеҗҺиө·зҡ„еҗҚеӯ—гҖӮMartin Fowler зҡ„иҜ„и®әж–Үз« зҢңжөӢзҒөж„ҹеҸҜиғҪжқҘиҮӘ Mitchell Hashimoto зҡ„еҚҡе®ўгҖӮдёҚз®ЎеҮәеӨ„еҰӮдҪ•пјҢиҝҷдёӘиҜҚиҝ…йҖҹиў«зӨҫеҢәжҺҘеҸ—пјҢеӣ дёәе®ғзІҫзЎ®ең°е‘ҪеҗҚдәҶдёҖд»¶еӨ§е®¶е·Із»ҸеңЁеҒҡдҪҶиҝҳжІЎжңүеҘҪеҗҚеӯ—зҡ„дәӢгҖӮ

д»Җд№ҲжҳҜ Harness

Harness зҡ„жң¬д№үжҳҜй©¬е…·вҖ”вҖ”зј°з»ігҖҒйһҚе…·гҖҒеҡјеӯҗвҖ”вҖ”дёҖж•ҙеҘ—з”ЁжқҘй©ҫй©ӯејәеЈ®дҪҶдёҚеҸҜйў„жөӢзҡ„еҠЁзү©зҡ„иЈ…еӨҮгҖӮйҡҗе–»еҫҲзӣҙзҷҪпјҡ

  • й©¬жҳҜ AI жЁЎеһӢпјҢеҠӣйҮҸеӨ§пјҢйҖҹеәҰеҝ«пјҢдҪҶиҮӘе·ұдёҚзҹҘйҒ“иҜҘеҫҖе“Әи·‘
  • й©¬е…·жҳҜеҹәзЎҖи®ҫж–ҪвҖ”вҖ”зәҰжқҹгҖҒжҠӨж ҸгҖҒеҸҚйҰҲеӣһи·ҜпјҢжҠҠеҠӣйҮҸеј•еҗ‘жңүз”Ёзҡ„ж–№еҗ‘
  • йӘ‘жүӢжҳҜдәәзұ»е·ҘзЁӢеёҲпјҢз»ҷж–№еҗ‘пјҢдёҚдәІиҮӘи·‘

жІЎжңү harness зҡ„ Coding Agent е°ұжҳҜдёҖеҢ№еңЁз©әж—·иҚүеҺҹдёҠж’’ж¬ўзҡ„зәҜиЎҖй©¬пјҡи·‘еҫ—жјӮдә®пјҢдҪҶе“Әе„ҝд№ҹеҲ°дёҚдәҶгҖӮ

Harness Engineering зҡ„жӯЈејҸе®ҡд№үеҸҜд»ҘжӢҶжҲҗеӣӣд»¶дәӢпјҡ

  1. зәҰжқҹвҖ”вҖ”Agent иғҪеҒҡд»Җд№ҲпјҲжһ¶жһ„иҫ№з•ҢгҖҒдҫқиө–и§„еҲҷпјү
  2. е‘ҠзҹҘвҖ”вҖ”Agent иҜҘеҒҡд»Җд№ҲпјҲдёҠдёӢж–Үе·ҘзЁӢгҖҒж–ҮжЎЈпјү
  3. йӘҢиҜҒвҖ”вҖ”Agent еҒҡеҜ№дәҶжІЎпјҲжөӢиҜ•гҖҒlintгҖҒCIпјү
  4. зә жӯЈвҖ”вҖ”еҒҡй”ҷдәҶжҖҺд№ҲеҠһпјҲеҸҚйҰҲеӣһи·ҜгҖҒиҮӘдҝ®еӨҚжңәеҲ¶пјү

е’Ң Prompt Engineering зҡ„еҢәеҲ«еңЁдәҺеұӮзә§пјҡprompt engineering е…іжіЁеҚ•ж¬ЎдәӨдә’зҡ„иҫ“е…ҘиҙЁйҮҸпјҢharness engineering е…іжіЁж•ҙдёӘзі»з»ҹвҖ”вҖ”зҺҜеўғгҖҒзәҰжқҹгҖҒеҸҚйҰҲгҖҒз”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶгҖӮдёҠдёӢж–Үе·ҘзЁӢпјҲcontext engineeringпјүжҳҜ harness зҡ„еӯҗйӣҶпјҢдёҚжҳҜе…ЁйғЁгҖӮ

жЁЎеһӢжҳҜйҖҡз”Ёе“ҒпјҢHarness жүҚжҳҜжҠӨеҹҺжІі

дёҖдёӘжңүиҜҙжңҚеҠӣзҡ„ж•°жҚ®зӮ№жқҘиҮӘ LangChainпјҡд»–д»¬зҡ„ Coding Agent еңЁ Terminal Bench 2.0 дёҠд»Һ 52.8% и·іеҲ° 66.5%пјҢжҺ’еҗҚд»Һ Top 30 и№ҝиҝӣ Top 5вҖ”вҖ”жЁЎеһӢжІЎжҚўпјҢеҸӘж”№дәҶ harnessгҖӮеҠ дәҶеҗҜеҠЁж—¶зҡ„зӣ®еҪ•з»“жһ„жҳ е°„гҖҒеҫӘзҺҜжЈҖжөӢгҖҒе®ҢжҲҗеүҚзҡ„иҮӘжЈҖжё…еҚ•гҖҒжҺЁзҗҶејәеәҰеҲҶеұӮпјҲи§„еҲ’е’ҢйӘҢиҜҒз”Ёй«ҳжҺЁзҗҶпјҢе®һзҺ°з”ЁдёӯжҺЁзҗҶпјүгҖӮеҗҢдёҖеҢ№й©¬пјҢжҚўеҘ—й©¬е…·пјҢиЎЁзҺ°еҲӨиӢҘдёӨдәәгҖӮ

OpenAI зҡ„е®һйӘҢжҳҜжӣҙжһҒз«Ҝзҡ„иҜҒжҚ®пјҡ3 еҗҚе·ҘзЁӢеёҲпјҲеҗҺжү©еҲ° 7 дәәпјүгҖҒ5 дёӘжңҲгҖҒзәҰ 1500 дёӘ PRгҖҒзәҰ 100 дёҮиЎҢд»Јз ҒгҖҒйӣ¶дәәе·Ҙзј–еҶҷгҖӮж•ҲзҺҮеӨ§зәҰжҳҜжүӢеҶҷзҡ„ 10 еҖҚгҖӮдә§е“ҒжңүеҮ зҷҫдёӘеҶ…жөӢз”ЁжҲ·еңЁж—ҘеёёдҪҝз”ЁгҖӮе·ҘзЁӢеёҲзҡ„е…ЁйғЁе·ҘдҪңе°ұжҳҜи®ҫи®Ў harnessвҖ”вҖ”е®ҡд№үж„ҸеӣҫгҖҒжҸҗдҫӣеҸҚйҰҲгҖҒиҖҢдёҚжҳҜеҶҷд»Јз ҒгҖӮ

дёүж №ж”Ҝжҹұ

з»јеҗҲ OpenAI зҡ„е®һи·өе’Ң Martin Fowler зҡ„еҪ’зәіпјҢHarness Engineering зҡ„ж ёеҝғз»“жһ„еҸҜд»ҘеҲҶдёәдёүеұӮгҖӮ

дёҠдёӢж–Үе·ҘзЁӢ

д»Һ Agent зҡ„и§Ҷи§’зңӢпјҢиҝҗиЎҢж—¶и®ҝй—®дёҚеҲ°зҡ„дёңиҘҝе°ұдёҚеӯҳеңЁгҖӮGoogle Docs йҮҢзҡ„еҶізӯ–гҖҒSlack йҮҢзҡ„и®Ёи®әгҖҒдәәи„‘дёӯзҡ„е…ұиҜҶвҖ”вҖ”е…ЁеңЁ Agent и§ҶйҮҺд№ӢеӨ–гҖӮд»“еә“йҮҢзүҲжң¬еҢ–зҡ„ж–Үд»¶жүҚжҳҜе®ғзҡ„е…ЁйғЁдё–з•ҢгҖӮ

OpenAI еӣўйҳҹж—©жңҹзҠҜиҝҮдёҖдёӘе…ёеһӢй”ҷиҜҜпјҡжҠҠжүҖжңүи§„еҲҷеЎһиҝӣдёҖдёӘе·ЁеӨ§зҡ„ AGENTS.mdгҖӮеӨұиҙҘеҺҹеӣ еҫҲзӣҙи§үвҖ”вҖ”дёҠдёӢж–ҮзӘ—еҸЈжҳҜзЁҖзјәе“ҒпјҢжҢҮд»Өж–Үд»¶еӨӘеӨ§дјҡжҠҠд»»еҠЎжҸҸиҝ°е’Ңд»Јз ҒжҢӨеҮәеҺ»пјӣд»Җд№ҲйғҪж ҮвҖқйҮҚиҰҒвҖқзӯүдәҺд»Җд№ҲйғҪдёҚйҮҚиҰҒпјӣеӨ§ж–Үд»¶иҝ…йҖҹи…җзғӮпјҢиҝҮж—¶и§„еҲҷе ҶжҲҗеқҹеңәгҖӮ

жңҖз»Ҳж–№жЎҲпјҡAGENTS.md еҺӢеҲ°зәҰ 100 иЎҢеҸӘеҪ“зӣ®еҪ•пјҢжҢҮеҗ‘ docs/ дёӢзҡ„з»ҶеҲҶж–ҮжЎЈгҖӮAgent д»ҺдёҖдёӘе°Ҹе…ҘеҸЈиҝӣжқҘпјҢиў«дёҖжӯҘжӯҘеј•еҗ‘иҜҘзңӢзҡ„ең°ж–№гҖӮз»ҷ Agent дёҖеј ең°еӣҫпјҢеҲ«з»ҷеҚғйЎөиҜҙжҳҺд№ҰгҖӮ

йқҷжҖҒдёҠдёӢж–Үд№ӢеӨ–пјҢеҠЁжҖҒдёҠдёӢж–ҮеҗҢж ·е…ій”®гҖӮOpenAI и®©еә”з”Ёж”ҜжҢҒжҢү git worktree еҗҜеҠЁзӢ¬з«Ӣе®һдҫӢпјҢжҠҠ Chrome DevTools еҚҸи®®жҺҘиҝӣ Agent иҝҗиЎҢж—¶пјҲAgent иғҪиҮӘе·ұж“ҚдҪңжөҸи§ҲеҷЁеӨҚзҺ° bugпјүпјҢз»ҷжҜҸдёӘ worktree йғЁзҪІдёҙж—¶еҸҜи§ӮжөӢжҖ§ж ҲпјҲAgent з”Ё LogQL жҹҘж—Ҙеҝ—гҖҒPromQL жҹҘжҢҮж ҮпјүгҖӮжңүдәҶиҝҷдәӣпјҢвҖңжңҚеҠЎеҗҜеҠЁеңЁ 800ms еҶ…вҖқиҝҷзұ»иҰҒжұӮе°ұдёҚеҶҚжҳҜдәәиӮүйӘҢж”¶пјҢиҖҢжҳҜ Agent иҮӘдё»йӘҢиҜҒзҡ„жҢҮж ҮгҖӮ

жһ¶жһ„зәҰжқҹ

е…үйқ ж–ҮжЎЈз»ҙжҢҒдёҚдәҶе…Ё Agent з”ҹжҲҗзҡ„д»Јз Ғеә“зҡ„дёҖиҮҙжҖ§гҖӮж ёеҝғзӯ–з•ҘпјҡзәҰжқҹдёҚеҸҳйҮҸпјҢдёҚз®Ўе®һзҺ°з»ҶиҠӮгҖӮ

OpenAI зҡ„еҒҡжі•жҳҜдёҘж јзҡ„еҲҶеұӮжһ¶жһ„вҖ”вҖ”жҜҸдёӘдёҡеҠЎеҹҹжҢү Types вҶ’ Config вҶ’ Repo вҶ’ Service вҶ’ Runtime вҶ’ UI еҲҶеұӮпјҢдҫқиө–ж–№еҗ‘еҚ•еҗ‘пјҢжЁӘеҲҮйҖ»иҫ‘пјҲи®ӨиҜҒгҖҒйҒҘжөӢгҖҒеҠҹиғҪејҖе…іпјүз»ҹдёҖиө° Providers жҺҘеҸЈгҖӮиҝҷдәӣзәҰжқҹйҖҡиҝҮиҮӘе®ҡд№ү linter е’Ңз»“жһ„жөӢиҜ•ејәеҲ¶жү§иЎҢпјҢlinter жң¬иә«д№ҹжҳҜ Codex з”ҹжҲҗзҡ„гҖӮдёҖдёӘе·§еҰҷзҡ„з»ҶиҠӮпјҡlint и§„еҲҷзҡ„жҠҘй”ҷдҝЎжҒҜйҮҢзӣҙжҺҘеҶҷдҝ®еӨҚжҢҮд»ӨпјҢAgent зңӢеҲ°жҠҘй”ҷзҡ„еҗҢж—¶е°ұжӢҝеҲ°дәҶжҖҺд№Ҳж”№зҡ„жҢҮеҚ—гҖӮ

иҝҷзұ»жһ¶жһ„жІ»зҗҶйҖҡеёёжҳҜеҮ зҷҫдәәзҡ„еӨ§еӣўйҳҹжүҚдјҡеҒҡзҡ„дәӢгҖӮеҜ№ Coding Agent жқҘиҜҙжҳҜж—©жңҹеүҚжҸҗвҖ”вҖ”жІЎжңүзәҰжқҹпјҢйҖҹеәҰи¶Ҡеҝ«жјӮз§»и¶ҠдёҘйҮҚгҖӮеңЁдәәзҡ„е·ҘдҪңжөҒйҮҢиҝҷдәӣи§„еҲҷеҸҜиғҪжҳҫеҫ—иҝӮи…җпјҢеҜ№ Agent жқҘиҜҙжҳҜд№ҳж•°ж•Ҳеә”пјҡеҶҷдёҖж¬ЎпјҢеӨ„еӨ„з”ҹж•ҲгҖӮ

дёҫдёӘе…·дҪ“дҫӢеӯҗпјҡеӣўйҳҹиҰҒжұӮеңЁж•°жҚ®иҫ№з•ҢеӨ„еҒҡж ЎйӘҢпјҲParse, donвҖҷt validateпјүпјҢдҪҶдёҚжҢҮе®ҡз”Ёе“ӘдёӘеә“гҖӮжЁЎеһӢиҮӘе·ұеҒҸеҘҪ ZodпјҢйӮЈе°ұ ZodгҖӮзәҰжқҹзҡ„жҳҜвҖқд»Җд№Ҳеҝ…йЎ»еҒҡвҖқпјҢдёҚжҳҜвҖқжҖҺд№ҲеҒҡвҖқгҖӮ

зҶөз®ЎзҗҶ

Agent дјҡеҝ е®һеӨҚзҺ°д»“еә“дёӯе·Іжңүзҡ„жЁЎејҸвҖ”вҖ”еҢ…жӢ¬дёҚеҘҪзҡ„гҖӮж—¶й—ҙдёҖй•ҝпјҢе‘ҪеҗҚжғҜдҫӢеҲҶжӯ§гҖҒж–ҮжЎЈдёҺд»Јз Ғи„ұиҠӮгҖҒжӯ»д»Јз Ғе Ҷз§ҜпјҢзҶөеўһдёҚеҸҜйҒҝе…ҚгҖӮ

OpenAI жңҖеҲқжҜҸе‘Ёдә”иҠұ 20% ж—¶й—ҙжүӢеҠЁжё…зҗҶвҖқAI ж®ӢжёЈвҖқпјҢеҫҲеҝ«жүӣдёҚдҪҸгҖӮжӣҝд»Јж–№жЎҲпјҡжҠҠвҖқй»„йҮ‘еҺҹеҲҷвҖқзј–з Ғиҝӣд»“еә“пјҢз”ЁеҗҺеҸ° Codex д»»еҠЎе®ҡжңҹжү«жҸҸеҒҸе·®гҖҒжӣҙж–°иҙЁйҮҸиҜ„еҲҶгҖҒжҸҗйҮҚжһ„ PRгҖӮеӨ§еӨҡж•°дёҖеҲҶй’ҹеҶ…е®Ўе®ҢиҮӘеҠЁеҗҲе№¶гҖӮеғҸеһғеңҫеӣһж”¶дёҖж ·пјҢжҢҒз»ӯе°ҸйўқиҝҳжҠҖжңҜеҖәпјҢж°ёиҝңеҘҪиҝҮж”’еҲ°зҲҶгҖӮ

Martin Fowler жҢҮеҮәиҝҷжҳҜжңҖиў«дҪҺдј°зҡ„дёҖзҺҜгҖӮжІЎжңүзҶөз®ЎзҗҶзҡ„ harness дјҡйҡҸж—¶й—ҙиЎ°иҙҘпјҢAgent ејҖе§ӢеӨҚеҲ¶и¶ҠжқҘи¶ҠеӨҡзҡ„еқҸжЁЎејҸпјҢеҪўжҲҗжҒ¶жҖ§еҫӘзҺҜгҖӮ

OpenAI е®һйӘҢдёӯзҡ„еҮ дёӘжңүи¶Јз»ҶиҠӮ

д»ҘдёҠдёүж №ж”ҜжҹұжҳҜзі»з»ҹжҖ§зҡ„жҖ»з»“пјҢOpenAI еҺҹж–ҮдёӯиҝҳжңүдёҖдәӣеҖјеҫ—еҚ•зӢ¬жӢҺеҮәжқҘзҡ„и§ӮеҜҹпјҡ

  • еҗһеҗҗйҮҸж”№еҸҳеҗҲе№¶е“ІеӯҰгҖӮPR з”ҹе‘Ҫе‘ЁжңҹеҫҲзҹӯпјҢеҒ¶еҸ‘жөӢиҜ•еӨұиҙҘйқ йҮҚи·‘и§ЈеҶіпјҢдёҚж— йҷҗжңҹйҳ»еЎһгҖӮAgent дә§еҮәйҖҹеәҰиҝңи¶…дәәзұ»е®ЎжҹҘеёҰе®ҪпјҢзә й”ҷжҲҗжң¬дҪҺпјҢзӯүеҫ…жҲҗжң¬й«ҳгҖӮдҪҺеҗһеҗҗйҮҸзҺҜеўғдёӢиҝҷд№ҲеҒҡдёҚиҙҹиҙЈд»»пјҢиҝҷдёӘзі»з»ҹйҮҢйҖҡеёёжҳҜеҜ№зҡ„гҖӮ
  • жҠҖжңҜйҖүеһӢеҒҸеҘҪвҖқж— иҒҠвҖқзҡ„жҠҖжңҜгҖӮAPI зЁіе®ҡгҖҒз»„еҗҲжҖ§еҘҪгҖҒеңЁи®ӯз»ғж•°жҚ®йҮҢе……еҲҶеҮәзҺ°зҡ„еә“еҜ№ Agent жӣҙеҸӢеҘҪгҖӮжңүж—¶и®© Agent иҮӘе·ұйҮҚеҶҷдёҖдёӘеҠҹиғҪеӯҗйӣҶпјҢжҜ”еј•е…ҘеӨ–йғЁеә“еҶҚз»•иҝҮй»‘з®ұиЎҢдёәжӣҙеҲ’з®—гҖӮ
  • еҚ•ж¬Ў Codex иҝҗиЎҢз»Ҹеёёи¶…иҝҮ 6 е°Ҹж—¶пјҢйҖҡеёёеңЁдәәзқЎи§үзҡ„ж—¶еҖҷгҖӮдәәзұ»жҺҢиҲөпјҢAgent жү§иЎҢвҖ”вҖ”еҢ…жӢ¬еңЁж·ұеӨңгҖӮ
  • Agent е·Із»ҸиғҪз«ҜеҲ°з«Ҝе®ҢжҲҗдёҖдёӘеҠҹиғҪпјҡжЈҖжҹҘд»Јз Ғеә“ вҶ’ йҮҚзҺ° bug вҶ’ еҪ•жј”зӨәи§Ҷйў‘ вҶ’ дҝ®еӨҚ вҶ’ йӘҢиҜҒ вҶ’ еҶҚеҪ•и§Ҷйў‘ вҶ’ ејҖ PR вҶ’ еӣһеә”еҸҚйҰҲ вҶ’ дҝ®жһ„е»ә вҶ’ еҸӘеңЁйңҖиҰҒеҲӨж–ӯж—¶еҚҮзә§з»ҷдәә вҶ’ еҗҲе№¶гҖӮдҪҶеӣўйҳҹиҜҡе®һең°иҜҙпјҢиҝҷй«ҳеәҰдҫқиө–зү№е®ҡд»“еә“зҡ„з»“жһ„е’Ңе·Ҙе…·пјҢеҲ«жҢҮжңӣејҖз®ұеҚіз”ЁеӨҚеҲ¶гҖӮ

Martin Fowler зҡ„е»¶дјёжҖқиҖғ

Martin Fowler еңЁиҜ„и®әж–Үз« йҮҢжҸҗдәҶеҮ дёӘжңүж„ҸжҖқзҡ„жҺЁжј”пјҡ

Harness дјҡдёҚдјҡеҸҳжҲҗж–°дёҖд»ЈжңҚеҠЎжЁЎжқҝпјҹ еӨ§еӨҡж•°з»„з»ҮеҸӘжңүдёӨдёүдёӘдё»иҰҒжҠҖжңҜж ҲгҖӮжңӘжқҘеӣўйҳҹеҸҜиғҪд»ҺдёҖз»„й’ҲеҜ№еёёи§Ғеә”з”ЁжӢ“жү‘зҡ„ harness жЁЎжқҝиө·жӯҘпјҢеҶҚж №жҚ®иҮӘе·ұзҡ„еңәжҷҜйҖҗжӯҘи°ғж•ҙвҖ”вҖ”е°ұеғҸд»ҠеӨ©зҡ„ golden path service templateпјҢдҪҶеҢ…еҗ«дәҶиҮӘе®ҡд№ү linterгҖҒз»“жһ„жөӢиҜ•гҖҒеҹәзЎҖдёҠдёӢж–Үж–ҮжЎЈе’ҢеҠЁжҖҒдёҠдёӢж–ҮжҺҘе…ҘгҖӮ

жҠҖжңҜж ҲдјҡдёҚдјҡеҠ йҖҹж”¶ж•ӣпјҹ еҪ“еҶҷд»Јз ҒеҸҳжҲҗвҖқеј•еҜј Agent з”ҹжҲҗд»Јз ҒвҖқпјҢжЎҶжһ¶е’Ң SDK зҡ„дәәдҪ“е·ҘеӯҰдҫқз„¶йҮҚиҰҒпјҲеҜ№дәәеҘҪз”Ёзҡ„дёңиҘҝеҜ№ Agent йҖҡеёёд№ҹеҘҪз”ЁпјүпјҢдҪҶејҖеҸ‘иҖ…зҡ„дёӘдәәеҒҸеҘҪдјҡи¶ҠжқҘи¶ҠдёҚйҮҚиҰҒгҖӮеӣўйҳҹеҸҜиғҪдјҡдјҳе…ҲйҖүжӢ©вҖқAI еҸӢеҘҪвҖқзҡ„жҠҖжңҜж ҲвҖ”вҖ”жңүзҺ°жҲҗ harnessгҖҒи®ӯз»ғж•°жҚ®е……и¶ігҖҒAPI зЁіе®ҡзҡ„йӮЈдәӣгҖӮ

Pre-AI д»Јз Ғеә“ vs. Post-AI д»Јз Ғеә“дјҡеҲҶеҢ–еҗ—пјҹ еҘҪзҡ„ harness жҠҖжңҜиғҪеӣһжәҜеә”з”ЁеҲ°иҖҒд»Јз Ғеә“еҗ—пјҹиҝҳжҳҜеҸӘеҜ№д»Һйӣ¶ејҖе§ӢгҖҒдёҖејҖе§Ӣе°ұжҢү harness жҖқи·Ҝжҗӯе»әзҡ„йЎ№зӣ®жңүж•ҲпјҹиҖҒд»Јз Ғеә“йҖҡеёёйқһж ҮеҮҶеҢ–гҖҒзҶөеҖјй«ҳпјҢзЎ¬еҘ— harness еҸҜиғҪеғҸеҜ№дёҖдёӘд»ҺжІЎи·‘иҝҮйқҷжҖҒеҲҶжһҗзҡ„йЎ№зӣ®йҰ–ж¬ЎејҖеҗҜ linterвҖ”вҖ”е‘ҠиӯҰж·№жІЎдёҖеҲҮгҖӮ

иў«еҝҪз•Ҙзҡ„зјәеҸЈпјҡFowler жҢҮеҮә OpenAI зҡ„ж–Үз« еҮ д№ҺжІЎж¶үеҸҠеҠҹиғҪе’ҢиЎҢдёәеұӮйқўзҡ„йӘҢиҜҒгҖӮд»–д»¬и®ІдәҶжһ¶жһ„зәҰжқҹгҖҒж–ҮжЎЈдёҖиҮҙжҖ§гҖҒд»Јз ҒйЈҺж јпјҢдҪҶвҖқиҝҷдёӘеҠҹиғҪжҳҜеҗҰзңҹзҡ„жҢүйў„жңҹе·ҘдҪңвҖқиҝҷдёӘй—®йўҳжІЎжңүзі»з»ҹжҖ§еӣһзӯ”гҖӮ
